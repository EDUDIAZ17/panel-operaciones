Política de Privacidad

Última actualización: 8 de junio de 2026

En Alexa Transportes (en adelante, "la Empresa"), operado bajo propiedad tecnológica de Nanometricasolucion, nos tomamos muy en serio la seguridad y privacidad de tus datos personales. Esta Política de Privacidad describe cómo recopilamos, utilizamos, almacenamos y protegemos la información a través de nuestro portal logístico y la integración con servicios de mensajería como la API de WhatsApp Cloud de Meta.

1. Información que Recopilamos

Nuestro sistema recopila únicamente los datos necesarios para la gestión operativa y de seguridad del transporte de carga, incluyendo:

  • Datos de contacto de Operadores y Personal: Nombres completos y números de teléfono móvil.
  • Datos de Ubicación Satelital (GPS): Datos de localización en tiempo real de las unidades de transporte (obtenidos mediante la integración autorizada con la API de Samsara).
  • Información del Viaje: Puntos de origen, destino, itinerario, tiempos de carga/descarga y comentarios de seguridad.

2. Uso de la Información

La información recopilada se utiliza exclusivamente para fines operativos y de seguridad vial:

  • Notificaciones Automáticas: Enviar avisos de seguridad y reportes de llegada/salida a los operadores y personal logístico a través de WhatsApp.
  • Monitoreo de Geocercas: Evaluar distancias de aproximación para alertar al área de Torre de Control cuando una unidad ingresa al destino.
  • Historial Operativo: Registrar bitácoras del viaje y firmas de tiempo con fines de auditoría logística interna.

3. Almacenamiento y Protección de Datos

Todos los datos se almacenan de forma segura en bases de datos cifradas proporcionadas por Supabase. Implementamos medidas técnicas, administrativas y físicas estrictas para proteger la información contra accesos no autorizados, pérdida, alteración o divulgación. No vendemos ni compartimos datos personales con terceros para fines publicitarios.

4. Derechos del Usuario (ARCO)

Como titular de tus datos personales, puedes ejercer tus derechos de Acceso, Rectificación, Cancelación y Oposición (Derechos ARCO) enviando una solicitud formal a nuestro correo electrónico de contacto.
